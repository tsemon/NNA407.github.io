### 보건통계학1 - 숙제 4

#### Q1. $$\beta $$ -blocker

###### 문제조건
- $$n_1 = 30$$, $$n_2 = 30$$, $$\bar{x_1}=65.2$$, $$s_1 = 7.8$$, $$\bar{x_2} = 70.3$$, $$s_2=8.3$$ 

##### (가) 가설을 설정하고 t-검정을 실시하여라. 유의 수준은 5%

###### 가설
- $$\beta$$-blocker는 심장수술 환자의 맥박을 감소시키는 영향이 있다. ($$H_1(\equiv H_0) = 65.2$$, $$H_2 (\equiv H_A) \neq 65.2$$)

###### t-검정
- 첫 표본과 두 번째 표본의 크기의 합, $$n_1 + n_2 \geq 30$$이고 공통분산 $$\sigma^2$$가 알려져 있지 않은 경우 공통분산 $$\sigma^2$$는 표본 공통분산 $$s_p^2$$로 대치된다.

	$$s_p^2 = \frac{(n_1 -1)s_1^2 + (n_2 -1 )a_2^2 }{(n_1 + n_2 -2 )} = \frac{(30-1)7.8^2+(30-1)8.3^2}{(30+30-2)}\approx64.865$$

- 따라서, 위 두 값의 z-value는

	$$z = \frac{\bar{x_1}-\bar{x_2}}{\sqrt{\sigma^2 (1/n_1 + 1/n_2)}} \\ = \frac{(65.2-70.3)}{\sqrt{(64.865(1/30 + 1/30)}} \approx -2.45251$$
    
	로 정의 된다. 

- 따라서 위 z-value에 해당하는 $$p(z) = 0.007093$$이며, 이는 설정한 $$\alpha = 0.05$$보다 작으므로 두 값은 유의하다.

- 그러므로, "$$\beta$$-blocker는 심장수술 환자의 맥박을 감소시킨다."



##### (나) 군 간 평균 차이의 95% 신뢰구간 제시.

###### 95% 신뢰수준 하에서 $$(\bar{x_1}-\bar{x_2})$$의 신뢰구간
- 95% 신뢰수준 하에서의 신뢰구간은 다음과 같다.
	
    $$(\bar{x_1}-\bar{x_2}) \pm z\sqrt{\frac{s_1 ^2}{n_1}+\frac{s_2^2}{n_2}} \\= (65.2 - 70.3) \pm 1.96 \sqrt{\frac{7.8^2}{30}+\frac{8.3^2}{30}} \approx (-9.17583, -1.02417)$$
    
    
#### Q2. 모유수유/분유가 유아의 철분결핍에 미치는 영향

##### 문제조건
- $$n_1 = 23$$, $$n_2 = 19$$, $$\bar{x_1}=13.3$$, $$\bar{x_2} = 12.4$$, $$s_1 = 1.7$$, $$s_2 = 1.8$$

##### (가) 모유수유/분유 그룹의 평균 헤모글로빈 수치가 다른지 검정하여 보아라.

###### 귀무가설/ 대립가설
- $$H_0$$ : 모유수유와 분유섭취는 유아의 철분결핍에 영향을 미치지 않는다.
- $$H_1$$ : 모유수유와 분유섭취는 유아의 철분결핍에 영향을 미친다.
- $$H_0 = 13.3$$, $$H_1\neq 13.3$$

###### t-test

- 첫 표본과 두 번째 표본의 크기의 합, $$n_1 + n_2 = 41 (\geq 30)$$이고 공통분산 $$\sigma^2$$가 알려져 있지 않은 경우 공통분산 $$\sigma^2$$는 표본 공통분산 $$s_p^2$$로 대치된다.

	$$s_p^2 = \frac{(n_1 -1)s_1^2 + (n_2 -1 )a_2^2 }{(n_1 + n_2 -2 )} = \frac{(23-1)1.7^2+(19-1)1.8^2}{(23+19-2)}\approx 3.0475$$

- 따라서, 위 두 값의 z-value는

	$$z = \frac{\bar{x_1}-\bar{x_2}}{\sqrt{\sigma^2 (1/n_1 + 1/n_2)}} \\ = \frac{(13.3-12.4)}{\sqrt{(3.0475(1/23 + 1/19)}} \approx 1.66298$$
    
	로 정의 된다. 

- 따라서 위 z-value에 해당하는 $$p(z) = 0.04816$$이며, 이는 $$\alpha = 0.05$$보다 작으므로 두 값은 유의하다.

###### 결론
- 모유수유와 분유섭취의 차이는 유아의 철분결핍에 영향을 미치며, 분유섭취는 유아의 철분 결핍을 발생시킨다.

##### (나) 사용한 가정에 대해 기술하라.
- 두 모집단은 정규분포를 따른다 (정규성).
- 표본들이 서로 다른 평균을 갖는 모집단에서 추출되었더라도, 분산이 동일하다고 가정한다. (등분산성)
- 두 표본의 크기의 합이 큰 경우, 공통분산 $$\sigma^2$$는 공통분산 $$s_p^2$$로 대치할 수 있다.



#### Q3. 운동이 우울증에 미치는 영향

##### (가) 분산분석표를 채워라.

| Source | Sum of Squares | Degrees of freedom | Mean Squares | F | p-value |
|:--------:|:--------:|:--------:|:--------:|:--------:|:--------:|
| Between Groups | 476.88 | 3 | 158.96 | 2.6011 | 0.072 |
| Within Groups  | 2261.16 | 37 | 61.1124 | | |
| Total | | 39 | | | | |
 
- $$k$$ is the number of groups, $$n$$ is the number of members in group.
- $$a(\equiv SST) = MST \times (k-1) = 158.96 \times 3 = 476.88$$.
- $$b = k-1 = 4-1 = 3$$.
- $$c = n-k = 40-3 = 37$$.
- $$d = n-1 = 40-1 = 39$$.
- $$e(\equiv MSE) = SSE / (n-k) \\= 2261.16 / (40-3) \approx 61.1124$$.
- $$f(\equiv MST/MSE) = 158.96 / 61.1124 \approx 2.6011$$.

##### (나) 귀무가설과 대립가설
- 귀무가설 : 운동은 우울증에 영향을 미치지 않는다. (모든 집단의 평균은 같다.)
- 대립가설 : 운동은 우울증에 영향을 미친다. (적어도 집단의 평균은 같지 않다.)

##### (다) 귀무가설이 참일 때 F통계량의 분포는 무엇인가?
- 귀무가설이 참인 경우, F통계량은 자유도가 1, n-2인 분포를 갖는다.

##### (라) 검정결과를 요약하고 해석하라.
- 4개의 그룹, 40의 n에 대한 ANOVA이다.
- $$F \geq F_{3,37}$$ 또는 p-value가 유의수준 보다 작은 경우 귀무가설을 기각한다.
	- $$F = 2.6011$$, $$F_{3,30} = 2.92$$, $$F_{3,40} = 2.84$$로 $$F \geq F_{3,37}$$을 만족하지 못한다.
	- $$\alpha(\equiv 0.072) > 0.05$$으로 유의수준보다 큰 값을 갖는다.
- 따라서, 운동은 우울증에 영향을 미치지 않는다는 귀무가설은 기각되지 않는다.
